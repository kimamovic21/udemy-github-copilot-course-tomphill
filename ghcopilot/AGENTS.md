# Custom GitHub Copilot Agents (ghcopilot)

## @lesson-guide

**Description**: Course navigation and learning assistant

**Capabilities**:

- Explain GitHub Copilot features covered in each lesson
- Provide step-by-step guidance for course exercises
- Generate practice prompts for Copilot
- Help understand inline suggestions vs chat features

**Usage**: Type `@lesson-guide` in Copilot Chat

**Example queries**:

- `@lesson-guide what should I learn in section 2?`
- `@lesson-guide explain inline suggestions`
- `@lesson-guide give me practice exercises`

---

## @code-reviewer

**Description**: Review and improve code quality for course projects

**Capabilities**:

- Review JavaScript code for best practices
- Check HTML semantic structure
- Validate CSS organization and basic theming
- Suggest accessibility improvements
- Generate meaningful commit messages

**Usage**: Type `@code-reviewer` in Copilot Chat

**Example queries**:

- `@code-reviewer check my script.js file`
- `@code-reviewer suggest accessibility improvements`
- `@code-reviewer generate a commit message for my changes`

---

## How to Use This File

1. Invoke agents in Copilot Chat by typing `@agent-name` followed by your question
2. Customize agents by editing their descriptions and capabilities
3. Add new agents by following the same structure above
