# Custom GitHub Copilot Agents

## @bucks2bar-helper

**Description**: Specialized assistant for the bucks2bar financial tracker application

**Capabilities**:

- Help with Chart.js configuration and customization
- Assist with Bootstrap 5 styling and responsive design
- Guide on implementing new financial tracking features
- Troubleshoot theme switching and dark mode issues
- Suggest improvements for data visualization
- Help with Jest testing setup and writing unit tests
- Debug authentication and login validation issues
- Assist with localStorage session management

**Usage**: Type `@bucks2bar-helper` in Copilot Chat followed by your question

**Example queries**:

- `@bucks2bar-helper how do I add a new chart type?`
- `@bucks2bar-helper fix the mobile responsive layout`
- `@bucks2bar-helper add export to CSV functionality`
- `@bucks2bar-helper help me write tests for the authentication function`
- `@bucks2bar-helper debug failing Jest tests`
- `@bucks2bar-helper improve test coverage for calculations`

## @lesson-guide

**Description**: Course navigation and learning assistant

**Capabilities**:

- Explain GitHub Copilot features covered in each lesson
- Provide step-by-step guidance for course exercises
- Generate practice prompts for Copilot
- Help understand inline suggestions vs chat features

**Usage**: Type `@lesson-guide` in Copilot Chat

**Example queries**:

- `@lesson-guide what should I learn in section 2?`
- `@lesson-guide explain inline suggestions`
- `@lesson-guide give me practice exercises`

---

## @code-reviewer

**Description**: Review and improve code quality for course projects

**Capabilities**:

- Review JavaScript code for best practices
- Check HTML semantic structure
- Validate CSS organization and dark theme implementation
- Suggest accessibility improvements
- Generate meaningful commit messages
- Review test coverage and suggest missing tests
- Analyze Jest test quality and structure
- Identify untested edge cases

**Usage**: Type `@code-reviewer` in Copilot Chat

**Example queries**:

- `@code-reviewer check my script.js file`
- `@code-reviewer suggest accessibility improvements`
- `@code-reviewer generate a commit message for my changes`
- `@code-reviewer analyze test coverage report`
- `@code-reviewer review my Jest test structure`
- `@code-reviewer identify missing test cases`

## How to Use This File

1. **Invoke agents** in Copilot Chat by typing `@agent-name` followed by your question
2. **Customize agents** by editing their descriptions and capabilities
3. **Add new agents** by following the same structure above
4. **Experiment** with different agent types for different project needs
